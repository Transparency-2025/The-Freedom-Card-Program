This **Launch Checklist** provides the immediate, concrete steps required to move from planning documents (A through J) to a fully operational program.

## ✅ Program Launch Checklist: The Beara Drive Initiative

This checklist is divided into three phases, starting with essential internal administration and leading to the public launch at Scoil Phobail Bhéara.

---

### Phase 1: Foundation & Administration (Internal Setup)

| Status | Action Item | Responsibility | Notes / Reference Document |
| :--- | :--- | :--- | :--- |
| ☐ | **Confirm Final Program Name** | Coordinator/School Board | Select from options (e.g., The Beara Drive Initiative). |
| ☐ | **Appoint Program Coordinator** | Principal/School Board | Designate the staff member responsible for tracking (Document I). |
| ☐ | **Secure Initial Funding** | Coordinator | Finalize pledges from Sponsors (Document D). |
| ☐ | **Finalize Budget Allocation** | Coordinator/Finance | Determine the exact value of Tier 1, 2, and 3 rewards. |
| ☐ | **Print/Digitize Administration Tools** | Coordinator/Admin | Print Student Charters (Document F) and set up the digital Tracking Sheet (Document I). |

---

### Phase 2: Logistics & Financial Platform (The Card Mechanism)

This phase is critical, as the choice of provider dictates the retailer experience.

| Status | Action Item | Responsibility | Notes / Reference Document |
| :--- | :--- | :--- | :--- |
| ☐ | **Select Gift Card Provider** | Coordinator/Finance | Choose between Local Town Card, One4All Partnership, or Restricted Prepaid Visa (Swirl/Perx). |
| ☐ | **Establish Retailer Opt-In Network** | Coordinator/Volunteers | Circulate the Retailer One-Pager (Document B) and register participating local businesses. |
| ☐ | **Design & Order Cards/Vouchers** | Coordinator | Incorporate the "Beara Advantage" logo and instructions on the card sleeve. |
| ☐ | **Set Up Merchant Category Code (MCC) Restrictions** | Card Provider | Crucial step to restrict spending to local retailers (MCCs for Retail, Fuel, Hospitality, etc.). |
| ☐ | **Load Initial Inventory** | Coordinator/Finance | Load the first batch of cards with funds from secured sponsorships. |

---

### Phase 3: Public Launch & Enrollment (External Engagement)

| Status | Action Item | Responsibility | Notes / Reference Document |
| :--- | :--- | :--- | :--- |
| ☐ | **Notify and Thank Sponsors** | Coordinator | Send the Partner Accountability Memo (Document J) to all donors. |
| ☐ | **Issue Press Release** | Coordinator/PR Officer | Distribute the Announcement (Document G) to *The Southern Star* and local media. |
| ☐ | **Present to Student Body** | Principal/Coordinator | Hold a school assembly to launch the program and explain the Student Charter (Document F). |
| ☐ | **Open Application Window** | Coordinator | Begin accepting signed Student Commitment Charters. |
| ☐ | **Distribute Student Welcome Pack** | Admin | Issue the welcome pack containing the program summary and contact details. |

***

This checklist provides a clear, sequential path to operationalising the program.
